Release notes

Version 0.3

Corrections et améliorations

  • Lecture du chmap de l'heure de "Dpart" : corrig, tant que la date et l'heure figurent bien dans le champ "formule" de la feuille excel et que la valeur est bien compressible par excel.
  • Le champ "Dure" est automatiquement recalcul si les champs "Dpart" ou "Arrive" sont modifis.

Version 0.2

Corrections et améliorations

  • Amélioration de la lecture des fichiers excels : les fichiers enregistrés depuis a priori n'importe quelle source (Mac, Windows, LibreOffice...) peuvent tre lus.
  • Le tableau de données peut tre placé n'importe o dans le classeur, le nom "Type" figurant le coin supérieur gauche du tableau.
  • Le contenu des données peut tre modifié directement dans le programme.
  • Corrections de bugs liés à un nombre de requtes trop importantes et problmes de synchronisation.

Note : l'API a l'air de se débrouiller aussi avec des noms de ville et pays en franais. Rester prudent toutefois.

Améliorations possibles :

Options pouvant tre choisie par l'utilisateur : temps à attendre entre les requêtes, nombre d'années à moyenner...
